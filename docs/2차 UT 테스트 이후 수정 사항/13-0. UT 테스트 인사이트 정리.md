## 🔬 2차 UT 종합 인사이트 정리

### ✅ 해소된 1차 이슈 (공통)

1. **자동수집 즉시 수집 오해 (1차 P0)** — 두 참가자 모두 시간 설정 UI를 즉시 인지. 손건우 주임님은 "굳이 설명이 필요 없을 정도"라고 평가[[1]](https://www.notion.so/26-03-26-DayZero-UT-2-_-c9c0b4e4858248c1a2bcf7606537dcb6?pvs=21)
2. **등록 후 수정/재업로드 부재 (1차 N13)** — 두 참가자 모두 수정 기능 발견에 어려움 없음. 홍상혁 팀장님: "변동알림에서 바로 수정할 수 있으니까 좋다"[[2]](https://www.notion.so/26-03-26-DayZero-UT-2-_-7be0a032b0df4212b7085948966cc4c9?pvs=21)
3. **수집→편집→등록 흐름** — 홍상혁 팀장님: "링크만 넣으면 수집되고, 편집하고 바로 판매에 입력하면 되니까 과정이 간단"[[2]](https://www.notion.so/26-03-26-DayZero-UT-2-_-7be0a032b0df4212b7085948966cc4c9?pvs=21)

### ❌ 미해소 / 신규 이슈 (공통 — 가장 중요)

1. **"변동알림중" 설정 진입점 UI** — **세션 전체 최대 실패 지점**. 두 참가자 모두 독자적 발견 실패. 손건우 주임님도 "콜아웃 클릭하는 건 이해하기가 어렵다"고 지적[[2]](https://www.notion.so/26-03-26-DayZero-UT-2-_-7be0a032b0df4212b7085948966cc4c9?pvs=21)[[1]](https://www.notion.so/26-03-26-DayZero-UT-2-_-c9c0b4e4858248c1a2bcf7606537dcb6?pvs=21)
2. **역마진 → 가격 수정 바로가기 부재** — 손건우 주임님: "역마진이어서 수정하기로 갔는데 가격 설정이 바로 나오지 않는다"[[1]](https://www.notion.so/26-03-26-DayZero-UT-2-_-c9c0b4e4858248c1a2bcf7606537dcb6?pvs=21)
3. **수정 페이지에 판매 일시중지/재개 버튼 없음** — 홍상혁 팀장님이 세부 페이지에서 자연스럽게 일시중지를 시도함[[2]](https://www.notion.so/26-03-26-DayZero-UT-2-_-7be0a032b0df4212b7085948966cc4c9?pvs=21)

### 🟡 부분 해소 (추가 검증 필요)

- **온보딩 스텝 형식** — 두 참가자 반응이 **엇갈림**
    - 손건우 주임님: "순서대로 할 수 있어서 직관적, 초반에 헷갈리는 게 덜 하다" (긍정)[[1]](https://www.notion.so/26-03-26-DayZero-UT-2-_-c9c0b4e4858248c1a2bcf7606537dcb6?pvs=21)
    - 홍상혁 팀장님: "설정은 쉬운데 정보를 찾으러 돌아가야 한다" (트레이드오프 지적)[[2]](https://www.notion.so/26-03-26-DayZero-UT-2-_-7be0a032b0df4212b7085948966cc4c9?pvs=21)
    - → **스텝 + 단일 화면 하이브리드** 검토 필요
- **편집화면 개선** — "편의성이 좋아졌다", "다 AI라 쉬웠다" 총평이나, 개별 UI 요소(상태 태그, AI 작성 안내, 자동 저장) 검증 데이터 부족[[2]](https://www.notion.so/26-03-26-DayZero-UT-2-_-7be0a032b0df4212b7085948966cc4c9?pvs=21)[[1]](https://www.notion.so/26-03-26-DayZero-UT-2-_-c9c0b4e4858248c1a2bcf7606537dcb6?pvs=21)
- **배송대행사 중복 선택 구조** — 손건우 주임님 세션에서만 확인. 여전히 두 곳에서 선택 가능해 혼란 지속. "한쪽에서만 선택하게 하고 안내문구만" 해결안 제시[[1]](https://www.notion.so/26-03-26-DayZero-UT-2-_-c9c0b4e4858248c1a2bcf7606537dcb6?pvs=21)

### 💡 핵심 차별점 (경쟁사 대비)

| 차별점 | 근거 |
| --- | --- |
| **AI 번역/편집** | 홍상혁 팀장님 1순위 차별점. 기존에는 "AI로 번역 후 포토샵으로 이미지 편집"까지 필요했으나 자동화됨 |
| **자동 수집** | 손건우 주임님 1순위 차별점. 2회 연속 핵심 전환 동기로 확인 |
| **과정의 단순함** | 두 참가자 공통. "링크 넣고 편집하고 등록" 3단계 흐름이 기존 도구 대비 간결 |

### 🔥 사용자 요청/니즈 (우선순위 순)

1. **역마진 AI 자동 가격 조정** — 홍상혁 팀장님 마무리 1순위 지목. 조건: AI 판매량 분석 + 경쟁사 가격 고려 + **사용자 ON/OFF 선택 필수**[[2]](https://www.notion.so/26-03-26-DayZero-UT-2-_-7be0a032b0df4212b7085948966cc4c9?pvs=21)
    - 원가 상승 시 → 판매가 자동 인상
    - 원가 하락 시 → 판매 성과 기반 분기 (잘 팔리면 유지, 안 팔리면 인하)
2. **가격 시뮬레이션 상시 표시** — 손건우 주임님: "항상 킨 상태로 오른쪽에 있으면 좋겠다"[[1]](https://www.notion.so/26-03-26-DayZero-UT-2-_-c9c0b4e4858248c1a2bcf7606537dcb6?pvs=21)
3. **반자동 수집 (크롬 익스텐션)** — 체크박스 선택 후 일괄 전송, 10~20개 제한[[2]](https://www.notion.so/26-03-26-DayZero-UT-2-_-7be0a032b0df4212b7085948966cc4c9?pvs=21)
4. **일괄 수정 기능** — 단건 편집에서도 다수 필드 한번에 수정[[1]](https://www.notion.so/26-03-26-DayZero-UT-2-_-c9c0b4e4858248c1a2bcf7606537dcb6?pvs=21)
5. **마스터/부관리자 권한 구분** — 팀 운영 셀러의 역할 기반 접근 제어[[1]](https://www.notion.so/26-03-26-DayZero-UT-2-_-c9c0b4e4858248c1a2bcf7606537dcb6?pvs=21)
6. **변동 알림 실시간 푸시** (슬랙/카카오톡)[[2]](https://www.notion.so/26-03-26-DayZero-UT-2-_-7be0a032b0df4212b7085948966cc4c9?pvs=21)

### ⚠️ 다음 세션에서 반드시 검증해야 할 항목

두 세션 모두에서 **디스커션 가이드 핵심 질문 다수 누락**으로 미확인된 항목들입니다:

- **B-2, B-3**: 자동수집 설정 완료 피드백 + 현재 상태 표시 (홍상혁 팀장님 세션)
- **C-1, C-3, C-5**: 편집화면 상태 태그 + AI 작성 안내 + 자동 저장 인지 (두 세션 모두)
- **D-4, D-5**: 소싱/Q10 링크 + 변동 히스토리 (두 세션 모두)
- **A-2, A-3**: 모달 디자인 혼동 + 개발 가이드 링크 (두 세션 모두)
