# 서비스 개요

---

## DayZero란?

한국 개인 셀러가 국내 쇼핑몰(올리브영, 쿠팡, 다이소 등)의 상품을 **Qoo10 JP(일본)에 자동으로 등록·판매**할 수 있게 해주는 크로스보더 SaaS.

**핵심 가치**: 소싱 → 편집 → 등록 파이프라인 자동화로, 초보 셀러도 일본 판매를 시작할 수 있다.

---

## 타겟 사용자

**한국 개인 셀러 (초보~중급)**
- Qoo10 JP를 통해 일본에 상품을 판매하려는 개인 사업자
- K-뷰티, K-팝 굿즈, 생활용품 등 한국 인기 상품 취급
- 일본어 능력 없음 → AI 번역 의존
- 기존 크로스보더 툴의 정보 과부하에 좌절한 경험

**셀러 성장 단계:**
| 단계 | 행동 패턴 | DayZero 지원 |
|------|-----------|-------------|
| 초보 | 첫 등록이 목표, 뭘 해야 할지 모름 | 온보딩 가이드 + 자동 수집 + AI 번역 |
| 중급 | URL로 특정 상품 수집, 가격 최적화 | 직접 수집 + 마진 시뮬레이션 |
| 숙련 | 대량 등록 + 가격/재고 모니터링 | 일괄 처리 + 자동 변동 확인 |

---

## 핵심 기능 파이프라인

```
수집 → 편집 → 등록 → 관리
```

### 1. 수집 (Sourcing)
국내 쇼핑몰에서 상품 정보를 자동으로 가져온다.

- **자동 수집**: 쇼핑몰 + 카테고리 + 시간 설정 → 매일 자동 실행
- **직접 수집**: URL 입력 → 즉시 수집 (최대 20건)
- **크롬 익스텐션**: 쇼핑몰 브라우징 중 원클릭 수집

### 2. 편집 (Editing)
수집된 상품을 Qoo10 JP에 맞게 가공한다.

- **AI 번역**: 상품명, 설명, 옵션을 한국어 → 일본어 자동 번역
- **카테고리 매핑**: Qoo10 카테고리 AI 자동 매핑
- **가격 자동 계산**: 원가 + 배송비 + 마진 + 수수료 → 판매가 자동 산출
- **이미지 번역**: 썸네일/상세이미지 내 텍스트 AI 번역

### 3. 등록 (Registration)
편집 완료된 상품을 Qoo10 JP에 등록한다.

- **개별 등록**: 편집 상세에서 바로 등록
- **일괄 등록**: 편집 목록에서 여러 상품 한 번에 등록
- **등록 결과**: 성공/실패 실시간 확인

### 4. 관리 (Monitoring)
등록 후 상품의 가격/재고 변동을 추적한다.

- **가격·품절 자동 확인**: 소싱처 원가 변동, 품절 감지
- **자동 대응**: 가격 자동 조정, 품절 시 판매 자동 중지
- **알림**: 역마진, 품절, 재입고 시 즉시 알림

## 프로토타입 범위

이 문서 세트는 **UT(사용성 테스트)용 프로토타입**을 기준으로 작성되었습니다.

- 모든 데이터는 더미 데이터 (실제 API 연동 없음)
- RPA 크롤링, Qoo10 API, 환율 API, AI 번역 API는 시뮬레이션
- **목적**: UX 흐름 검증 — 초보 셀러가 수집→편집→등록까지 막힘 없이 완주할 수 있는가
- 3회 UT를 통해 검증·보완 완료

프로덕션 개발 시 이 문서의 UX 흐름을 기준으로, 실제 API를 연동하여 구현합니다.
