## 7. 애니메이션 (ANIM)

`import { ANIM } from '@/design/animations'`으로 사용.
`<style>{ANIM.fadeInUp}</style>` 형태로 주입.

| 키 | 키프레임 | 용도 |
|----|---------|------|
| `fadeInUp` | translateY(8px) → 0, opacity | 리스트, 카드, 콜아웃 진입 |
| `fadeIn` | opacity 0 → 1 | 오버레이, 배경 |
| `slideUp` | translateY(20px) → 0 | 모달 콘텐츠 |
| `modalSlideUp` | translateY(16px) scale(0.98) → 0 scale(1) | 모달 + 스케일 |
| `tooltipFadeIn` | translateY(4px) → 0 | 툴팁 |
| `pageIn` | translateY(12px) → 0 | 페이지 진입 |
| `spin` | rotate(0) → rotate(360deg) | 로딩 스피너 |
| `rowSlideIn` | translateY(8px) → 0 | 리스트 행 |
| `calloutIn` | translateY(-8px) → 0 | 콜아웃 (위→아래) |
| `koIn` | translateY(-4px) → 0 | 한국어 원문 표시 |
| `toastBubbleUp` | translateY(12px) scale(0.9) → translateY(0) scale(1), opacity 0 → 1 | 토스트 메시지 진입 (0.4s ease) |

## 8. 공통 UI 패턴

### 버튼

**Primary (CTA):**
- 배경: `colors.primary`, 호버: `colors.primaryDark`
- 텍스트: `colors.white`, `font.weight.semibold`
- 높이: 52px (온보딩), 48px (일반), 44px (보조)
- border-radius: `radius.lg`
- press 효과: `scale(0.98)`

**Secondary:**
- 배경: `colors.bg.surface`, 테두리: `colors.border.default`
- 텍스트: `colors.text.secondary`
- 호버: 배경 `colors.bg.page`

**Ghost (텍스트 링크):**
- 배경: transparent
- 텍스트: `colors.text.tertiary`, 호버 시 `colors.text.primary`

**Danger:**
- 배경: `colors.danger`
- 텍스트: `colors.white`

**비활성:**
- 배경: `colors.border.default` 또는 `colors.bg.subtle`
- 텍스트: `colors.text.muted`
- cursor: `not-allowed`

### 인풋

- 패딩: `spacing['4']` (16px)
- border: `1.5px solid colors.border.default`
- border-radius: `radius.lg` (온보딩) 또는 `radius.md` (일반)
- 포커스: border `colors.primary` + box-shadow `0 0 0 3px rgba(49, 130, 246, 0.1)`
- 에러: border `colors.danger`
- 비활성: 배경 `colors.bg.subtle`
- 폰트: `font.size.base`, `colors.text.primary`
- placeholder: `colors.text.muted` 또는 `colors.text.placeholder`

### 카드

- 배경: `colors.bg.surface`
- 테두리: `colors.border.default` 1px
- border-radius: `radius.xl` (16px)
- 그림자: `shadow.sm` 또는 없음
- 패딩: `spacing['6']`~`spacing['8']`

### 탭

**세그먼트 컨트롤 (SC-01):**
- 컨테이너: `colors.bg.subtle` 배경, `radius.lg`, `spacing['1']` 패딩
- 활성 탭: `colors.bg.surface` 배경, `shadow.sm`, `font.weight.bold`
- 비활성: transparent, `colors.text.muted`

**언더라인 탭 (ED-01, ED-02, RG-01):**
- 하단 2px 보더
- 활성: `colors.primary` 보더, `colors.primary` 텍스트, `font.weight: 700`
- 비활성: transparent 보더, `colors.text.tertiary`

### 토글 스위치

- 크기: 44x24px
- ON: `colors.primary` 배경, 원형 노브 right 22px
- OFF: `colors.border.default` 배경, 원형 노브 left 2px
- 노브: 20x20px, 흰색, `shadow.sm`
- transition: 0.2s

### 필터 칩

- 활성: `colors.primaryLight` 배경, `colors.primary` 텍스트, `font.weight.semibold`, 체크 아이콘
- 비활성: `colors.bg.surface` 배경, `colors.border.default` 테두리, `colors.text.tertiary`
- border-radius: `radius.full`

### 뱃지

- 배경: `colors.bg.subtle` (기본) / `colors.primaryLight` (활성) / `colors.dangerBg` (에러)
- border-radius: `radius.full`
- 폰트: `font.size['2xs']`~`font.size.xs`, `font.weight.bold`
