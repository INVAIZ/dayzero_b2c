# [S-02] Qoo10 JP API 연동

## 화면 목적
셀러의 Qoo10 JP 스토어 계정을 API Key로 연동한다. 연동 완료 시 스토어 정보를 확인시켜 준다.

## 진입 경로
- S-01 로그인/회원가입 성공 후 (`/qoo10-connect`)
- S-03에서 스텝 1 클릭 (온보딩 스텝 네비게이션)

## 소스 파일
- `pages/onboarding/Qoo10ConnectPage.tsx` — 페이지 전체
- `components/onboarding/OnboardingLayout.tsx` — 공통 레이아웃 + 스텝 인디케이터
- `components/onboarding/OnboardingContext.tsx` — 상태 관리
- `components/onboarding/useOnboardingTransition.ts` — 페이지 전환 애니메이션
- `mock/authMock.ts` — `mockQoo10Connect()` (1.5초 대기)

## OnboardingLayout 스텝 인디케이터 상세

**스텝 원형 (36x36):**

| 상태 | 배경 | 테두리 | 내용 | 그림자 |
|------|------|--------|------|--------|
| 완료 | `colors.primary` | 없음 | `Check` 아이콘 (16px, 흰색) | 없음 |
| 현재 | `colors.bg.surface` | `2.5px solid colors.primary` | 숫자 (파란색) | `0 0 0 4px colors.primaryHover` + pulse 애니메이션 |
| 미방문 | `colors.bg.subtle` | `2px solid colors.border.default` | 숫자 (회색) | 없음 |

**현재 스텝 펄스 애니메이션 (`subtlePulse`):**
- `2.5s ease-in-out infinite`
- `0%, 100%`: `box-shadow: 0 0 0 4px colors.primaryHover`
- `50%`: `box-shadow: 0 0 0 6px colors.primaryHover`

**연결선 (80x3px):**
- 배경: `colors.border.default`
- 완료 시: `colors.primary`로 채워짐 (`width: 0% → 100%`, `0.6s cubic-bezier(0.16, 1, 0.3, 1)`)

**클릭 가능 조건:** 이미 방문한 스텝만 클릭 가능 (`maxVisited` 추적)
- 호버: 원형 `scale(1.1)` + `box-shadow`, 라벨 `colors.primary`
- 액티브: `scale(0.95)`

**콘텐츠 전환 애니메이션:**
- 진입: `opacity 0→1`, `translateY(16px→0)`, `0.5s cubic-bezier(0.16, 1, 0.3, 1)`
- 퇴장 (`exiting`): `opacity 1→0`, `translateY(0→-12px)`, `0.3s cubic-bezier(0.4, 0, 1, 1)`
- 프로그레스 바 전환 후 콘텐츠 표시 (같은 스텝이면 50ms, 다른 스텝이면 650ms 대기)

**페이지 전환 (`useOnboardingTransition`):**
- `transitionTo(path)` 호출 → `exiting = true` → **300ms** 후 `navigate(path)`

## 동작 정의

### API Key 연동
- Given: API Key 입력됨
- When: 폼 제출 (버튼 클릭 또는 Enter)
- Then: 빈 값 체크 → 로딩 → `mockQoo10Connect(apiKey)` → 완료 상태
  - 응답: `{ storeName: 'デイゼロ東京セレクト', sellerId: 'QA_test_seller' }`

### 빈 입력 제출
- Given: API Key 비어있음 (공백만 포함)
- When: 제출
- Then: `"API Key를 입력해주세요"` 에러 표시 (빨간 테두리 + 에러 텍스트)

### 입력 시 에러 클리어
- Given: 에러 표시 중
- When: 입력 필드 값 변경
- Then: 에러 클리어

### 다른 계정 연동
- Given: 연동 완료 상태
- When: "다른 계정으로 연결하기" 클릭
- Then: `connected = false` → 미연동 상태 UI (apiKey 유지)

### 다음 단계
- Given: 연동 완료
- When: "다음 단계로 계속" 클릭
- Then: `transitionTo('/basic-info')` → 300ms 퇴장 애니메이션 → S-03 이동
