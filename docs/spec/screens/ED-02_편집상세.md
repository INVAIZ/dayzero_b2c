# [ED-02] 상품 편집 상세

## 화면 목적
개별 상품의 번역/편집을 수행한다. 기본 정보, 가격, 이미지를 탭으로 나누어 편집.
편집 완료 후 바로 Qoo10 등록도 가능.

## 진입 경로
- ED-01 상품 행 클릭
- URL 직접 접근 (`/editing/:productId`)

## 레이아웃
Sidebar 직접 포함 (MainLayout 미사용, 자체 레이아웃)

```
┌──────────┬──────────────────────────────────────────┐
│          │ [← 목록으로] | [썸네일+상품명] [◀ N/M ▶] │
│ Sidebar  │ [소싱처링크]              [삭제] [등록]   │
│          ├──────────────────────────────────────────┤
│          │ [기본 정보] [가격 정보] [썸네일/상세페이지] │
│          │                                          │
│          │ ── 탭 콘텐츠 영역 (max-width: 760px) ──   │
│          │                                          │
└──────────┴──────────────────────────────────────────┘
```

## 헤더 (EditingHeader) — sticky 고정

- "← 목록으로" 버튼 → `/editing` 이동
- 상품 썸네일 (40x40) + 소싱처 로고 (16x16)
- 상품명 표시:
  - 번역 완료: 일본어 상품명 (밑줄 점선, 호버 시 한국어 원문 툴팁)
  - 번역 전: 한국어 상품명
- 소싱처 원본 링크 (ExternalLink 아이콘, 새 탭)
- 이전/다음 상품 이동 (◀ ▶, "N / M" 표시)
- "삭제" 아이콘 (Trash2, 호버 시만 표시)
- "등록하기" 버튼:
  - 번역 완료 시: `colors.primary` 배경, 활성
  - 번역 미완료 시: 비활성 + **호버 툴팁** "상품명과 상세설명 번역이 완료되면 등록할 수 있어요."

## 탭 구조

| 탭 | 키 | 내용 |
|----|----|------|
| 기본 정보 | `basic` | 상품명, 설명, 옵션, 카테고리, 브랜드, 발송 설정 |
| 가격 정보 | `price` | 원가, 비용 구조, 판매가, 예상 수익 |
| 썸네일/상세페이지 | `images` | 썸네일 이미지 + 상세페이지 이미지 |

- URL `?tab=` 파라미터로 초기 탭 지정 가능
- 활성 탭: 하단 2px `colors.primary` 보더, `font.weight: 700`

## 기본 정보 탭 (BasicEditTab) — 필드별 상세

### 자동 저장

모든 편집은 **2초 디바운스 자동 저장**:
1. 필드 변경 감지
2. 2초 대기 (추가 변경 시 리셋)
3. "저장 중..." 표시 (해당 섹션 옆)
4. 저장 완료 → "저장됨 ✓" 표시 (2초 후 사라짐)

### 일괄 AI 편집 버튼

상단에 "일괄 AI 편집" 버튼:
- 클릭 시 미완료 항목을 순차적으로 자동 번역 (상품명 → 옵션 → 설명)
- 진행 표시: "1/3 상품명을 번역하고 있어요..." → "2/3 상품 옵션을 번역하고 있어요..." → "3/3 상세설명을 번역하고 있어요..."
- 이미 완료된 단계는 건너뜀

### 섹션 1: 상품명

```
┌──────────────────────────────────────────────────┐
│ 상품명 *           [상태 태그]         [AI 번역]  │
│                                                  │
│ (한국어 원문 — 접기/펼치기)                        │
│ ┌──────────────────────────────────────────────┐ │
│ │ 일본어 상품명 입력 (textarea)                  │ │
│ └──────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────┘
```

**상태별 분기:**

| 상태 | 상태 태그 | 인풋 스타일 | AI 번역 버튼 |
|------|-----------|------------|-------------|
| 미번역 (titleJa 없음) | 주황 "번역 필요" (⚠ Info 아이콘) | **좌측 3px 주황 보더** (`colors.warningIcon`) | 활성 "AI 번역" |
| AI 번역 중 | — | **셰이머 인풋**: 파란 배경(`colors.primaryLight`) + Loader2 스피너 + "번역하고 있어요..." (그라디언트 텍스트) | 비활성 "처리 중..." |
| 번역 완료 | 파란 "AI 번역" (✓ Check 아이콘) | 일반 인풋 (주황 보더 해제) | 활성 (재번역 가능) |
| 한국어 포함 (수동 편집 후) | 주황 "번역 필요" | 좌측 주황 보더 재표시 | 활성 "AI 번역" |

**한국어 원문 표시:**
- 기본: 접혀 있음 (접두사 제거된 한국어 상품명)
- 클릭 시 펼침/접기
- 펼침 애니메이션: `koIn 0.2s`

**인풋 포커스/블러:**
- 포커스: 보더 `colors.primary`, 좌측 주황 보더도 파란색으로 전환
- 블러: 미번역이면 좌측 주황 보더 복원

### 섹션 2: 상세설명

```
┌──────────────────────────────────────────────────┐
│ 상세설명 *         [상태 태그]                     │
│ [AI 작성] [AI 번역]  [한국어|일본어 토글]          │
│                                                  │
│ ┌──────────────────────────────────────────────┐ │
│ │ 상세설명 textarea                              │ │
│ │                                    N / 10,000 │ │
│ └──────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────┘
```

**2단계 프로세스:** AI 작성 (한국어) → AI 번역 (일본어)

| 동작 | 조건 | 결과 |
|------|------|------|
| **AI 작성** 클릭 | 설명 비어있음 | 셰이머 (3.5~5초) → 한국어 설명 생성 → `descMode: 'ko'` |
| **AI 작성** 클릭 | 설명 이미 있음 | 확인 모달 "이미 작성된 내용이 있어요. 새로 작성하면 기존 내용이 대체됩니다." |
| **AI 번역** 클릭 | 한국어 설명 있음 | 셰이머 (2.5~3.5초) → 일본어 번역 생성 → `descMode: 'ja'` |
| **AI 번역** 클릭 | 한국어 없고 일본어에 한국어 포함 | 일본어 내용을 한국어 원본으로 복사 후 번역 |
| **AI 번역** 호버 | 한국어 원본 없음 | 툴팁 "먼저 AI 작성으로 한국어 원본을 만들어주세요" |

**한국어/일본어 토글:**
- 한국어 탭 선택: 한국어 원본 표시 (편집 가능)
- 일본어 탭 선택: 일본어 번역본 표시 (편집 가능)
- 글자수 카운터: "N / 10,000" (최대 10,000자)

**상태 태그:**
- 미작성: 주황 "번역 필요"
- 한국어만 있음: 주황 "번역 필요"
- 일본어 완료: 파란 "완료" (Check 아이콘)
- 일본어에 한국어 포함: 주황 "번역 필요" + 경고 "한국어가 포함되어 있어요"

**셰이머 상태 (AI 작성/번역 중):**
- textarea 영역 전체가 파란 배경 + 셰이머 그라디언트
- Loader2 스피너 + "작성하고 있어요..." 또는 "번역하고 있어요..." (그라디언트 텍스트)

### 섹션 3: 옵션

```
┌──────────────────────────────────────────────────┐
│ 옵션 (N개)        [상태 태그]   [일괄 AI 번역]    │
│                                                  │
│ [이미지] [일본어 옵션명 인풋]   [재고] [삭제]      │
│ [이미지] [일본어 옵션명 인풋]   [재고] [삭제]      │
│ [이미지] [셰이머 — 번역 중...]  [재고] [삭제]      │
│                                                  │
│ [+ 옵션 추가]                                    │
└──────────────────────────────────────────────────┘
```

**옵션별 상태:**

| 상태 | 인풋 | 표시 |
|------|------|------|
| 미번역 (nameJa 없음, nameKo에 한국어 포함) | 좌측 3px 주황 보더 | placeholder에 한국어 원문 |
| AI 번역 중 | 파란 배경 + Loader2 + "번역하고 있어요..." | 셰이머 그라디언트 텍스트 |
| 번역 완료 | 일반 인풋 | `content-fade-in` 애니메이션 |
| 번역 완료인데 한국어 포함 | 좌측 주황 보더 + 하단 경고 | "한국어가 포함되어 있어요. 한국어를 지우거나 번역해 주세요." |

**일괄 AI 번역:**
- 미번역 옵션만 대상 (이미 번역 완료된 옵션은 건너뜀)
- 순차 처리: 각 옵션 1.2~1.8초 간격
- 각 옵션이 완료될 때마다 즉시 인풋에 반영 + 셰이머 해제

**옵션 삭제:**
- 삭제 아이콘 (Trash2) 클릭 → 확인 모달
- 제목: "옵션을 삭제할까요?"
- 설명: ""{옵션명}"을 삭제하면 복구할 수 없습니다."

```
┌──────────────────────────────────────────────────┐
│ 브랜드                        [상태 뱃지] [변경]  │
```

**매칭 상태별 뱃지:**

### 섹션 6: 기타 정보

- **제조사** (`manufacturer`): 읽기 전용
- **원산지** (`productionPlace`): 읽기 전용
- **발송가능일**: 2개 드롭다운
  - 발송 유형: 일반발송 / 당일발송 / 예약발송 (4일 이상)
  - 입금일 기준: 유형별 선택지
    - 일반발송: 1일/2일/3일 이내 (영업일 기준)
    - 예약발송: 4일~14일 이내 (주말 포함)
    - 당일발송: 드롭다운 숨김
  - 예약발송 안내 텍스트:
    - "※ 4일~14일 이내는 주말포함 일수로 카운트되며, 예약설정으로 취급됩니다."
    - "※ 4일~14일은 배송포인트 플러스 점수가 부여되지 않습니다."
    - "※ 4일~14일은 상품+옵션금액 기준 2% 수수료가 부과됩니다."
  - 일반발송 안내: "※ 1~3일 이내는 영업일수로 카운트 됩니다. (토/일 휴무 제외)"

**SourceTag (출처 태그):**
각 필드 옆에 데이터 출처를 표시:

| 출처 | 태그 | 색상 | 호버 툴팁 |
|------|------|------|----------|
| `crawled` | Globe 아이콘 + 초록 배경 | 초록 | "실제 상품 페이지에서 수집한 {항목} 정보입니다." |
| `ai` | AI 아이콘 + 파란 배경 | 파란 | "상품 {항목} 정보가 없어 AI가 예측한 {항목}입니다." |
| `manual` | 연필 아이콘 + 회색 배경 | 회색 | "직접 입력한 {항목}입니다." |

**플로팅 툴팁:** 마우스 커서 따라다니는 툴팁 (뷰포트 밖으로 나가지 않도록 자동 보정)

**편집 가능 필드:**
- 상품 원가: 클릭 → 인풋 전환 → ₩ 입력 → 블러 시 저장
- 무게: 클릭 → 인풋 전환 → kg 입력 → 블러 시 저장 + 해외 배송비 재계산
- 마진율: 슬라이더 드래그 또는 직접 입력 → 즉시 재계산 (표시는 정수, 내부 저장은 소수)
- 옵션별 원가: 클릭 → 인풋 전환 → ₩ 입력 → 블러 시 저장 (0일 때 빈 입력란)
- 옵션별 판매가: 클릭 → 인풋 전환 → ¥ 입력 → 블러 시 저장 (대표 옵션 수정 시 마진율 역산)

**대표 옵션 설정:**
- 기본 정보 탭 옵션 섹션에 라디오 버튼으로 대표 옵션 선택
- 대표 변경 시 확인 모달: "'옵션명' 옵션을 대표 상품으로 설정하시겠습니까? 대표 상품의 판매가가 Qoo10에 노출됩니다."
- 가격 탭에서는 대표 뱃지 표시만 (변경 불가)

**수익 계산 부호 규칙:**
- 투입 비용 항목: 앞에 `−` 표시 (색상은 text.primary 유지, 빨간색 아님)
- 판매 정산 판매가: 앞에 `+` 표시
- 정산금액: 앞에 `+` 표시
- Qoo10 수수료: 앞에 `−` 표시 (danger 색상)

## 썸네일/상세페이지 탭 (images)

### 썸네일 (ThumbnailEditTab)

- 이미지 그리드 (최대 10장, [0]이 대표)
- 각 이미지 상태:
  - `none`: 원본 이미지
  - `processing`: AI 처리 중 (반투명 오버레이 + Loader2 스피너)
  - `completed`: 번역됨 (체크 뱃지)
- "AI 번역" 버튼: 이미지 내 텍스트 한→일 번역
- 배경 제거 토글

### 상세페이지 (DetailImageEditTab)

- 이미지 리스트 (최대 30장)
- 동일한 AI 번역/배경 제거 기능

## 동작 정의

### AI 번역 — 상품명
- Given: 기본 정보 탭, 일본어 상품명 비어있거나 한국어 포함
- When: "AI 번역" 클릭
- Then:
  1. 인풋이 셰이머 상태로 전환 (파란 배경 + "번역하고 있어요..." 그라디언트)
  2. 2.5~3.5초 후 일본어 번역 결과 입력
  3. 상태 태그 → "AI 번역" (파란, Check)
  4. 인풋 좌측 주황 보더 해제
  5. 자동 저장 트리거

### AI 작성 → 번역 — 상세설명
- Given: 기본 정보 탭, 상세설명 비어있음
- When: "AI 작성" 클릭
- Then:
  1. textarea 셰이머 상태 (3.5~5초)
  2. 한국어 상세설명 생성 → 한국어 탭으로 전환
  3. 한국어 내용 확인/수정 가능
- When: "AI 번역" 클릭 (한국어 있는 상태)
- Then:
  1. textarea 셰이머 (2.5~3.5초)
  2. 일본어 번역 생성 → 일본어 탭으로 전환
  3. `translationStatus` → 'completed'

### 옵션 개별 번역
- Given: 옵션 행
- When: (일괄 AI 번역 실행 시 또는 개별 처리)
- Then: 해당 옵션 인풋만 셰이머 → 1.2~1.8초 → 일본어 결과

### 카테고리 변경
- Given: 기본 정보 탭
- When: "변경" 클릭
- Then: CategorySelectModal 열림 (대분류 → 중분류 → 소분류)

### 등록하기
- Given: 텍스트 번역 완료 (`isTextTranslated`)
- When: 헤더 "등록하기" 클릭
- Then: 등록 프로세스 시작 → `/registration` 이동

### 삭제
- Given: 편집 상세
- When: 삭제 아이콘 클릭
- Then: 확인 모달
  - 제목: "이 상품을 삭제할까요?"
  - 설명: "수집 목록에서 제거되고, 편집 내용도 함께 삭제돼요. 이 작업은 되돌릴 수 없어요."
- When: 확인
- Then: 삭제 + `/editing` 이동

### 이전/다음 상품 이동
- When: ◀ / ▶ 클릭
- Then: 해당 상품으로 URL 변경
- 모든 로컬 상태 초기화 (번역 중 상태, 저장 상태 등)

### 키보드 단축키
- `Ctrl+Z` (INPUT/TEXTAREA 외): `/editing` 이동

## 검증 체크리스트

### 기본 구조
- [ ] 상품 정보 정확히 로드 + 상품 전환 시 모든 로컬 상태 초기화
- [ ] 3개 탭 전환 + URL ?tab= 파라미터
- [ ] 헤더 sticky + 이전/다음 이동
- [ ] 존재하지 않는 productId → /editing 리다이렉트

### 상품명
- [ ] 미번역: 좌측 주황 보더 + "번역 필요" 태그
- [ ] AI 번역 클릭 → 셰이머 (파란 배경 + 그라디언트 텍스트) → 결과 입력
- [ ] 번역 완료: 주황 보더 해제 + "AI 번역 ✓" 태그
- [ ] 한국어 원문 접기/펼치기
- [ ] 포커스 시 보더 색상 전환

### 상세설명
- [ ] AI 작성 → 셰이머 → 한국어 생성 → 한국어 탭 전환
- [ ] AI 번역 → 셰이머 → 일본어 생성 → 일본어 탭 전환
- [ ] 내용 있을 때 AI 작성 → 덮어쓰기 확인 모달
- [ ] 한국어 없을 때 AI 번역 호버 → 툴팁
- [ ] 한국어/일본어 탭 전환
- [ ] 글자수 카운터 (N / 10,000)
- [ ] 일본어에 한국어 포함 시 경고

### 옵션
- [ ] 옵션별 미번역 주황 보더
- [ ] 일괄 AI 번역 → 순차 셰이머 → 완료
- [ ] 번역 후 한국어 포함 경고
- [ ] 옵션 삭제 확인 모달
- [ ] 옵션 추가

### 일괄 AI 편집
- [ ] 미완료 항목만 순차 처리 (상품명→옵션→설명)
- [ ] 진행 상태 표시 "N/M" + 단계별 텍스트

### 카테고리/브랜드
- [ ] 카테고리 변경 → 3단계 모달
- [ ] 브랜드 매칭 상태 뱃지 (matched/unmatched/none)
- [ ] 브랜드 변경 모달

### 가격
- [ ] 비용 구조 + SourceTag 출처 표시
- [ ] SourceTag 호버 → 플로팅 툴팁
- [ ] 원가/무게 편집 → 즉시 재계산
- [ ] 마진율 슬라이더/직접 입력 → 판매가 재계산

- [ ] 썸네일/상세이미지 표시
- [ ] AI 번역 + 배경 제거
