# [ED-01] 편집 목록

## 화면 목적
수집된 상품의 번역/편집 상태를 한눈에 파악하고,
다음 행동(AI 번역, 개별 편집, 일괄 등록, 삭제)을 선택하는 허브 화면.

## 진입 경로
- 사이드바 "수집된 상품" 클릭
- 수집 완료 알림 클릭 (`?focusJobId={jobId}` 파라미터로 해당 수집건 자동 선택)
- ED-02에서 "목록으로" 클릭

## 레이아웃
MainLayout

```
┌──────────────────────────────────────────────────────────────────┐
│ [전체 (N)] [편집 필요 (N)] [편집 완료 (N)]  ← 언더라인 탭       │
│                                                                  │
│ [소싱처 필터 칩들] [검색 인풋]                                     │
│                                                                  │
│ ☐ 전체선택  상품명▲▼  카테고리  무게  판매가▲▼  원가  수집일▲▼    │
│ ┌──────────────────────────────────────────────────────────────┐ │
│ │ ☐ [썸네일] [로고] 상품명  [상태칩]  카테고리  무게  ¥가격  ₩원가 │ │
│ │          [남은할일 칩]                                        │ │
│ ├──────────────────────────────────────────────────────────────┤ │
│ │ ☐ [썸네일] [로고] 상품명  [AI편집중]  ...                     │ │
```

## 진입 시 자동 동작
- 사이드바 "수집된 상품" 뱃지 카운트(`unprocessedProductCount`) 0으로 초기화
- 화면 이탈 시 모든 상품을 읽음 처리 (`markProductsRead`)
- 모든 상품의 판매가를 현재 온보딩 설정(마진율 등) 기준으로 재계산

## 상품 리스트 아이템 (ProductListItem) — 상태별 상세

### 행 구성 (컬럼)

```
[☐] [썸네일 48x48] [로고+상품명+상태칩] [카테고리] [무게] [판매가¥] [원가₩] [수집일]
                    [남은할일 칩 — 2행]
```

### 상태 1: 일반 상태 (번역 전)

- **상품명**: 한국어 원문 표시 (`titleKo`에서 접두사 제거)
- **상태 칩 없음**
- **남은 할일 칩** (주황색 배경):
  - 미완료 항목만 나열: "상품명 / 옵션 / 상세설명"
  - 예: 상품명만 미완료 → "상품명"
  - 전부 미완료 → "상품명 / 옵션 / 상세설명"
- **좌측 경고 보더 없음** (목록에서는 칩으로 표시)

### 상태 2: AI 편집 중 (processing)

- **행 전체**: `colors.bg.info` 배경 + 셰이머 오버레이 (반투명 파란 그라디언트 좌→우 반복)
- **체크박스**: 비활성 (opacity 0.3, 클릭 불가)
- **상품명**: 셰이머 텍스트 (그라디언트 애니메이션으로 반짝이는 텍스트)
- **상태 칩**: "AI 편집 중" (파란 배경, `Loader2` 스피너 11px)
- **행 클릭 불가** (cursor: default)
- **남은 할일 칩 숨김**

### 상태 3: 번역 완료 (일본어 상품명 있음)

- **상품명**: 일본어 번역본 표시 (`titleJa`에서 접두사 제거)
  - 밑줄 점선 (`text-decoration: underline dotted`)
  - **호버 시 툴팁**: "한국어 원문" 라벨 + 한국어 상품명 표시
  - 번역 직후 `fadeInUp 0.5s` 애니메이션
- **남은 할일 칩**: 필수 3개(상품명/옵션/설명) 중 미완료 항목만 표시

### 상태 4: 등록 준비 완료 (필수 편집 모두 완료)

- **남은 할일 칩**: 초록색 "등록 준비 완료" (Check 아이콘)
- 조건: `titleJa` 존재 + 모든 옵션 `nameJa` 존재 + `descriptionJa` 존재

### 상태 5: 번역 실패

- **상태 칩**: "번역 실패" (빨간 배경 `colors.dangerLight`, 빨간 텍스트)

### 호버 툴팁 (모든 상태 공통)

| 호버 대상 | 툴팁 내용 |
|-----------|----------|
| 상품명 (번역 완료 시) | "한국어 원문" + 한국어 상품명 |
| 카테고리 | "원본 카테고리" + 소싱처 카테고리 / "Qoo10 카테고리" + 매핑된 카테고리 |
| 무게 | 출처별 설명 (AI 추정 / 수집 / 직접 입력) |
| 판매가 (¥) | "한화 환산 (¥1 = ₩{환율})" + "약 ₩{환산액}" |
| 원가 (₩) | "엔화 환산 (₩1 = ¥{역환율})" + "약 ¥{환산액}" |
| 남은 할일 칩 | 체크리스트 (필수 3항목 + 선택 2항목, 각각 완료/미완료 표시) |

### 남은 할일 호버 체크리스트 상세

```
┌──────────────────────────────────────┐
│ ⓘ 등록하기 전에 편집이 필요해요.      │  ← 미완료 시
│ 또는                                 │
│ ⓘ 필수 편집은 완료! 추가 편집도 가능해요. │ ← 필수만 완료 시  
│ 또는                                 │
│ ⓘ 모든 편집이 완료되었어요!           │  ← 전체 완료 시
│ ──────────────────────────────────── │
│ [필수] 상품명 번역          ✅ / ⚠️   │
│ [필수] 옵션 번역            ✅ / ⚠️   │
│ [필수] 상세설명 작성 및 번역  ✅ / ⚠️   │
│ ──────────────────────────────────── │
│ 썸네일 이미지 번역           ✅ / ○   │
│ 상세페이지 번역              ✅ / ○   │
└──────────────────────────────────────┘
```

- 필수 미완료: ⚠️ 주황 아이콘 (`AlertCircle`)
- 필수 완료: ✅ 초록 체크 (`Check`)
- 선택 미완료: ○ 빈 원 (연한 테두리)
- 선택 완료: ✅ 초록 체크

### 무게 컬럼 상세

- **무게 있음**: "{N}kg" + 출처 태그
- **무게 없음**: "무게 설정 필요" (빨간 텍스트 `colors.danger`)
- **출처 태그** (작은 아이콘):
  - `crawled` (수집): Globe 아이콘, 초록 배경
  - `ai` (AI 추정): AI 아이콘, 파란 배경
  - `manual` (직접 입력): 회색 배경

## AI 편집 모달 (TranslationModal) — 상세

### 모달 구성

```
┌──────────────────────────────────┐
│ AI 편집 설정                 [X] │
│                                  │
│ ⚠️ 선택된 상품 중 N건은 이미     │  ← 이미 편집된 상품 포함 시만
│ 편집된 상품입니다. 계속하면       │
│ 재편집됩니다.                    │
│                                  │
│ [☑] [필수] 상품명 번역           │  ← 체크박스 + 라벨
│ [☑] [필수] 옵션 번역             │
│ [☑] [필수] 상세설명 작성 및 번역  │
│ [☐] 썸네일 번역                  │
│ [☐] 상세페이지 번역              │
│                                  │
│ [N건 AI 편집 시작]               │
└──────────────────────────────────┘
```

### 대상 항목별 상태

| 항목 | 기본 선택 | "필수" 뱃지 | 이미 완료된 경우 |
|------|-----------|------------|----------------|
| 상품명 번역 | ✅ ON | 있음 | 회색 비활성 + "이미 완료되었어요" 텍스트, 클릭 불가 |
| 옵션 번역 | ✅ ON | 있음 | 동일 |
| 상세설명 작성 및 번역 | ✅ ON | 있음 | 동일 |
| 썸네일 번역 | ☐ OFF | 없음 | 동일 |
| 상세페이지 번역 | ☐ OFF | 없음 | 동일 |

### 경고 배너

- **조건**: 선택된 상품 중 이미 편집 완료된 상품이 1건 이상 있고, 전체 선택이 2건 이상일 때
- **텍스트**: "선택된 상품 중 **N건**은 이미 편집된 상품입니다. 계속하면 재편집됩니다."
- **스타일**: `colors.warningLight` 배경, `colors.warningBorder` 테두리, `AlertCircle` 아이콘

### 시작 버튼

- 텍스트: "{N}건 AI 편집 시작"
- 선택된 대상 0개 → 버튼 비활성 (opacity 0.5)

## 상태별 분기

### 빈 상태 (상품 0건)
- EmptyState 컴포넌트 (→ 공통 패턴 G-05 참조)
- 텍스트: "수집된 상품이 없어요."

### 정상 (상품 있음)
- 탭별 건수 뱃지 (소싱처 필터 무관하게 계산)
- 등록 완료/진행 중 상품은 편집 목록에서 자동 제외

### 편집 완료 판정 기준
- `isFullyTranslated`: titleJa 존재 + descriptionJa 존재 + 모든 옵션의 nameJa 존재 (또는 한국어에 한글이 없는 경우)
- `isFullyTranslated`이 등록 가능 조건 (titleJa + descriptionJa + 모든 옵션 nameJa 존재)

### focusJobId 진입
- URL에 `?focusJobId={jobId}` 파라미터
- 탭을 "전체"로, 필터를 "전체"로 리셋
- 해당 jobId의 상품들 자동 선택
- 해당 요소로 smooth 스크롤
- URL에서 파라미터 제거 (replace)

## 동작 정의

### 탭 전환
- Given: 편집 목록에 있음
- When: "편집 필요" 탭 클릭
- Then: `isFullyTranslated`가 false인 상품만 필터링
- When: "편집 완료" 탭 클릭
- Then: `isFullyTranslated`가 true인 상품만 필터링

### 소싱처 필터
- Given: 탭 내 상품 목록
- When: 소싱처 칩 클릭
- Then: 해당 소싱처 상품만 표시
- 필터 칩은 **현재 탭에 존재하는 소싱처만** 동적으로 표시

### 검색
- Given: 검색 입력 필드
- When: 텍스트 입력
- Then: `titleKo` 또는 `titleJa`에 부분 일치하는 상품만 표시

### 정렬
- 정렬 가능 컬럼: 수집일 (`createdAt`), 판매가 (`salePriceJpy`), 상품명 (`title`)
- 클릭 시 오름차순/내림차순 토글 (같은 컬럼 재클릭)
- 다른 컬럼 클릭 시: 수집일은 내림차순 기본, 나머지는 오름차순 기본
- 정렬 헤더에 ▲▼ 표시 (활성 방향만 진하게)

### 상품 선택
- Given: 상품 리스트
- When: 체크박스 클릭
- Then: 선택 상태 토글 + 하단 일괄 작업 바 표시
- **AI 편집 중인 상품**: 체크박스 비활성 (선택 불가)
- "전체 선택": 현재 **페이지** 상품 전체 선택/해제

### 상품 클릭 (편집 상세 이동)
- Given: 상품 리스트에서 행 클릭 (체크박스 외)
- When: 클릭
- Then: `/editing/{productId}` (ED-02) 이동
- **AI 편집 중인 상품**: 클릭 불가 (cursor: default)

### AI 편집 (일괄)
- Given: 1건 이상 선택
- When: "AI 편집" 버튼 클릭
- Then: AI 편집 모달 표시 (위 모달 상세 참조)
- When: 모달에서 "N건 AI 편집 시작" 클릭
- Then:
  1. 모달 닫힘
  2. 토스트: "N건 번역을 시작했어요"
  3. 선택된 상품의 `translationStatus` → `processing`
  4. 해당 상품 행이 셰이머 상태로 전환
  5. 순차 처리 후 각 상품 `translationStatus` → `completed`
  6. 완료 시 상품명이 일본어로 바뀌며 `fadeInUp` 애니메이션

### 일괄 등록
- Given: 1건 이상 선택됨
- When: "등록하기" 클릭
- Then: 선택된 상품 중 `isFullyTranslated`가 false인 상품이 포함되어 있는지 확인

**분기 1: 모든 선택 상품이 `isFullyTranslated === true`**
→ 등록 프로세스 시작 → `/registration` 이동

**분기 2: 선택 상품 중 미번역(옵션 포함) 상품이 포함됨**
→ 확인 모달 표시:
- 제목: `번역이 완료되지 않은 상품이 있어요`
- 설명: `선택한 {총N}건 중 {미번역M}건은 옵션 번역이 완료되지 않아 등록할 수 없어요. 번역이 완료된 {등록가능K}건만 등록할까요?`
- 확인: `{K}건 등록하기` (K > 0일 때만 활성)
- 취소: `취소`
- K === 0일 때 (등록 가능한 상품이 없음): 확인 버튼 비활성 + 설명 텍스트 `"선택한 상품 모두 번역이 완료되지 않았어요. AI 편집을 먼저 실행해주세요."`

> 등록 가능 조건: `isFullyTranslated` (상품명 + 설명 + 모든 옵션 nameJa 존재)

### 일괄 삭제
- Given: 1건 이상 선택
- When: "삭제" 클릭
- Then: 확인 모달 표시
  - 제목: "선택한 N개의 상품을 삭제할까요?"
  - 설명: "삭제된 상품 정보는 복구할 수 없으며, 수집된 모든 데이터가 영구적으로 삭제됩니다."
- When: "삭제하기" 클릭
- Then: 선택 상품 삭제

### 페이지네이션
- 페이지당 20건
- 탭/필터/검색 변경 시 페이지 1로 리셋
