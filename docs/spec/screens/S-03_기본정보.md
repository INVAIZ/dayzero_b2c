### 가격 시뮬레이션 사이드 패널

**위치:** `position: absolute`, `left: calc(100% + 24px)`, `top: 0`, `width: 360px`
**스타일:** `colors.bg.subtle` 배경, `radius.xl`, `1px solid colors.border.default`
**스크롤:** `maxHeight: calc(100vh - 200px)`, `overflowY: auto`

**헤더:**
- `Calculator` 아이콘 (16px, `colors.primary`) + `"가격 시뮬레이션"` (`font.size.base`, `font.weight.bold`)

**환율 뱃지:**
- `colors.bg.info` 배경, `radius.full`, `1px solid colors.primaryLightBorder`
- `"오늘의 환율"` (`font.size.xs`, `font.weight.semibold`, `colors.primary`)
- `"¥1 = ₩{EXCHANGE_RATE}"` (`font.size.xs`, `font.weight.bold`, `colors.text.primary`)
- 현재 `EXCHANGE_RATE = 9.2`

**입력:**
- **상품 원가**: `₩` 접두어, `type="text"` (comma format), 기본값 `15,000`
- **무게 드롭다운** (`WeightDropdown`): `width: 120px`, `WEIGHT_OPTIONS = [0.1, 0.25, 0.3, 0.5, 0.75, 1.0, 1.5, 2.0, 2.5]`, 기본값 `0.3`
  - 배송비 산정: 프리셋 대행사 → `lookupShippingFee(forwarder, weight)`, 기타/미선택 → `intlShipping` 사용

**결과 테이블:**
- 배경: `colors.bg.page`, `radius.lg`, `1px solid colors.border.default`

| 행 | 라벨 | 값 |
|----|------|-----|
| 1 | `Qoo10 판매가` | `¥{finalPriceJpy}` (`colors.primary`, `font.weight.bold`) |
| 2 | `실수령액 (수수료 차감 후)` | `₩{payoutKrw}` (`colors.text.primary`, `font.weight.bold`) |
| 3 | `순수익` | `{+/-}₩{actualProfitKrw}` |

- 순수익 행: 양수 → `colors.success` + `colors.successBg` 배경, 음수 → `colors.danger` + `colors.dangerBg` 배경

**계산 과정 토글:**
- 버튼: `"계산 과정 보기"` / `"계산 과정 접기"` + 아이콘 (`ChevronRight` / `ChevronDown`)
- `1px solid colors.border.default`, `radius.md`, `colors.text.tertiary`
- 호버: `colors.bg.page` 배경

**계산 과정 펼침 시 (7행):**

| 아이콘 | 라벨 | 값 | highlight |
|--------|------|-----|-----------|
| `Package` | `원가 + 작업비` | `₩{totalCostKrw}` | — |
| `TrendingUp` | `+ 마진 {marginValue}%` | `₩{marginAmount}` | — |
| `Truck` | `+ 배송비` | `₩{shippingCostKrw} (¥{shippingJpy})` | — |
| `Calculator` | `+ Qoo10 수수료 12% 보전` | `¥{feeJpy}` | — |
| `Globe` | `= Qoo10 판매가` | `¥{finalPriceJpy}` | ✅ (`colors.primary`, `font.size.md`) |
| `RefreshCw` | `− 수수료 12% 차감 → 수령액` | `₩{payoutKrw}` | — |
| `Package` | `− 원가·작업비·배송비 → 순수익` | `₩{actualProfitKrw}` | ✅ (`colors.primary`, `font.size.md`) |

- 행 사이: `ArrowRight` 아이콘 (12px, `colors.border.default`, `rotate(90deg)`)
- 하단: `"Qoo10 수수료: 패션/뷰티 10~12% · 기타 6~10% (현재 12% 적용)"` (`font.size.xs`, `colors.text.muted`)

**가격 계산 로직:**
```
costKrw = simBaseCost + prepCost
desiredProfit = round(costKrw × marginValue / 100)
shippingCostKrw = round(shippingJpy × EXCHANGE_RATE)
requiredPayoutKrw = costKrw + desiredProfit + shippingCostKrw
requiredPayoutJpy = round(requiredPayoutKrw / EXCHANGE_RATE)
finalPriceJpy = round(requiredPayoutJpy / (1 - PLATFORM_FEE_RATE))
feeJpy = round(finalPriceJpy × PLATFORM_FEE_RATE)
payoutJpy = finalPriceJpy - feeJpy
payoutKrw = round(payoutJpy × EXCHANGE_RATE)
actualProfitKrw = payoutKrw - costKrw - shippingCostKrw
```

## 동작 정의

### 배송대행사 선택
- Given: 드롭다운 열림
- When: 대행사 선택 (예: 큐익스프레스)
- Then: 드롭다운 닫힘 + 해당 대행사 창고 주소 자동 입력 + 주소 필드 disabled

### "직접 입력" 선택
- Given: 드롭다운 열림
- When: "직접 입력하기 (기타)" 선택
- Then: 드롭다운 닫힘 + 주소 필드 빈값으로 리셋 + 편집 가능

### Step 1 → Step 2
- Given: 연락처 입력됨 (`isStep1Valid`)
- When: "다음" 클릭
- Then: Step 2 (출하지 주소) 표시

### Step 2 → Step 3
- Given: 출하지 + 반품 주소 유효 (`isStep2Valid`)
- When: "다음" 클릭
- Then: Step 3 (해외 배송비) 표시

### Step 3 → Step 4
- Given: Step 3
- When: "다음" 클릭
- Then: Step 4 (작업비) 표시

### Step 4 → Step 5
- Given: Step 4
- When: "다음" 클릭
- Then: Step 5 (마진율) 표시

### Step 5 → S-05
- Given: Step 5
- When: "다음 단계로 계속" 클릭
- Then: `transitionTo('/extension-install')` → S-05 이동

### 이전 스텝 이동
- Given: Step N (N > 1)
- When: "이전으로" 클릭
- Then: Step N-1 복귀

### Step 3 → Step 2 (대행사 변경)
- Given: Step 3
- When: "다른 배송대행사를 쓰시나요?" 클릭
- Then: Step 2 (출하지 주소) 복귀
