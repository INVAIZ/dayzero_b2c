# [ST-01] 설정

## 화면 목적
온보딩에서 설정한 판매 기본값(배송대행사, 주소, 마진율, 배송비, 환율)을 수정하고,
Qoo10 연동 상태 및 계정 정보를 확인한다.

## 진입 경로
- 사이드바 "설정" → 하위 메뉴 클릭 (판매 설정 / Qoo10 연동 / 계정)
- URL 직접 접근 (`/settings/:tab`)
- `/settings` → `/settings/sales`로 리다이렉트 (`<Navigate to="/settings/sales" replace />`)

## 소스 파일
- `pages/settings/SettingsPage.tsx` — 페이지 전체 (메인 + SalesContent + Qoo10Content + AccountContent + ApplyScopeModal + PriceSimulation)
- `components/layout/Sidebar.tsx` — 사이드바 + 설정 하위 메뉴 트리
- `components/onboarding/OnboardingContext.tsx` — 상태 관리 (state, setState)
- `utils/forwarder.ts` — `FORWARDER_PRESETS`, `FORWARDER_RATES`, `lookupShippingFee()`, `PLATFORM_FEE_RATE`, `EXCHANGE_RATE`, `WEIGHT_OPTIONS`

### 변경 감지 흐름
1. 페이지 마운트 시: `extractDraft(state)`로 `draft`와 `savedSnapshot` 초기화

3. 변경 판단: `getChangedKeys(draft, savedSnapshot)` — 키별 `!==` 비교
4. `hasChanges = changedKeys.length > 0`
5. `hasChanges`이면: 헤더에 "변경사항 있음" 뱃지 + "저장하기" 버튼 활성화

## 동작 정의

### 설정 저장 (비가격 필드)
- Given: 판매 설정 탭, 비가격 필드만 변경됨
- When: "저장하기" 클릭
- Then: 즉시 저장 + 토스트 `"설정이 저장되었어요"` (2.5초)

### 설정 저장 (가격 영향 필드)
- Given: 판매 설정 탭, 가격 영향 필드 변경됨
- When: "저장하기" 클릭
- Then: ApplyScopeModal 표시
- When: 항목/범위 선택 후 "저장하기" 클릭
- Then: 저장 + 토스트

### 미저장 이탈 (라우터)
- Given: `hasChanges === true`
- When: 다른 페이지 이동 시도
- Then: 이탈 확인 모달 표시
- "나가기": `blocker.proceed()` — 변경 폐기 후 이동
- "저장하고 나가기": 저장 처리 후 이동 (가격 필드 변경 시 ApplyScopeModal 먼저)

### 미저장 이탈 (브라우저)
- Given: `hasChanges === true`
- When: 브라우저 새로고침/닫기
- Then: `beforeunload` 기본 다이얼로그

### 배송대행사 변경
- Given: 판매 설정 탭
- When: 배송대행사 드롭다운 변경
- Then: 출하지 주소 자동 변경 (프리셋이면 자동 입력 + disabled, 기타면 빈 값 + 편집 가능) + draft 업데이트

### 환율 모드 전환
- Given: 판매 설정 탭
- When: 실시간/고정 라디오 변경
- Then: 콜아웃 박스 `settingsFadeIn 0.3s ease` 애니메이션으로 전환
- 실시간: `EXCHANGE_RATE` 상수 사용
- 고정: `draft.fixedExchangeRate` 사용

### 마진 슬라이더/입력 조작
- Given: 판매 설정 탭
- When: 슬라이더 드래그 또는 직접 입력 (5~60 범위)
- Then: 시그널 즉시 업데이트 (색상 + 아이콘 + 메시지) + 시뮬레이션 결과 실시간 재계산 (`useMemo`)

### 시뮬레이션 원가/무게 변경
- Given: 가격 시뮬레이션 섹션
- When: 원가 또는 무게 변경
- Then: 결과 즉시 재계산

### 해외 배송비 배너 클릭
- Given: 판매가 구성 섹션, 배송대행사 선택 상태
- When: "배송대행사 ... 요율표 자동 적용 중" 배너 클릭
- Then: `#forwarder-section`으로 smooth scroll
