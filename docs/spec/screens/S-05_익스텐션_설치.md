# [S-05] 크롬 익스텐션 설치

## 화면 목적
DayZero 전용 크롬 익스텐션(수집 프로그램) 설치를 안내한다.
설치하면 쇼핑몰에서 직접 상품을 수집할 수 있다. 건너뛰기 가능.

## 진입 경로
- S-03 기본 정보 완료 → "다음 단계로 계속"

## 소스 파일
- `pages/onboarding/ExtensionInstallPage.tsx` — 페이지 전체
- `components/onboarding/OnboardingLayout.tsx` — 공통 레이아웃
- `design/animations.ts` — `ANIM.fadeInUp`, `ANIM.modalSlideUp`, `ANIM.fadeIn`

---

## 레이아웃

OnboardingLayout, 현재 스텝: **3** (마지막).

3개 Phase로 구성: `idle` → `waiting` → `done`

---

## Phase 상태 관리

| 상태 | 타입 | 초기값 | 설명 |
|------|------|--------|------|
| `phase` | `'idle' \| 'waiting' \| 'done'` | `'idle'` | 현재 단계 |
| `showComplete` | `boolean` | `false` | 온보딩 완료 화면 표시 |

## Phase 1: idle (기본)

**진입 애니메이션:** `fadeInUp 0.4s ease both`

### 제목/설명

- 제목: `"빠른 상품 수집을 위해 프로그램 설치가 필요해요"`
  - `font.size['xl+']`, `font.weight.semibold`, `letterSpacing: -0.5px`
  - `wordBreak: keep-all`
- 설명: `"DayZero 전용 프로그램을 설치하면 쇼핑몰을 둘러보면서\n원하는 상품을 바로바로 담을 수 있어요."`
  - `font.size.base`, `colors.text.tertiary`, `font.weight.medium`
  - 애니메이션: `fadeInUp 0.4s ease 0.05s both` (0.05초 딜레이)

### 모션 일러스트

**컨테이너:**
- 너비: `100%`, 높이: `190px`
- 배경: `colors.bg.subtle`, `radius.xl`
- 테두리: `1px solid colors.border.default`
- `overflow: hidden`

**내부 구성:**
- 쇼핑몰 헤더: `"인기 상품 추천"` / `"더보기"`
- 상품 카드 3개 (가로 배치, 각 flex: 1)
- 장바구니 배지 (우하단)
- macOS 커서 (SVG 화살표)

**상품 카드 데이터:**

| # | 브랜드 | 상품명 | 할인율 | 가격 |
|---|--------|--------|--------|------|
| 0 | `닥터지` | `레드 블레미쉬 클리어 수딩 크림 70ml` | `32%` | `18,900원` |
| 1 | `라네즈` | `네오 쿠션 매트 15g 본품+리필 세트` | `28%` | `32,000원` |
| 2 | `코스알엑스` | `어드밴스드 스네일 96 에센스 100ml` | `41%` | `11,900원` |

**9초 CSS 애니메이션 사이클:**

모든 애니메이션은 `9s infinite` 반복.

**커서 이동 (`cursorPath`):**
- 0~4%: 좌하단에서 등장 (opacity 0→1)
- 12%: 카드 0 위치 도착
- 14%: 클릭 (scale 0.7)
- 15.5%: 바운스 (scale 0.85)
- 17%: 복귀 (scale 1)
- 38%: 카드 1 위치 도착 → 40% 클릭
- 64%: 카드 2 위치 도착 → 66% 클릭
- 88%: 퇴장 (opacity 0)
- 100%: 리셋

**카드 하이라이트 (`cardHL0/1/2`):**
- 커서 도착 시: `box-shadow: 0 0 0 2px colors.primary 60%`
- 클릭 시: `box-shadow + glow`, 배경 `colors.bg.info`
- 이후 서서히 복귀

**DayZero 수집 버튼 오버레이 (`ov0/1/2`):**
- 카드 좌상단 (8px, 8px)
- 커서 도착 전 등장 (scale 0.5→1)
- 22x22 파란 라운드 버튼, DayZero 아이콘 (13x13)
- 클릭 바운스 (`btnClick0/1/2`): scale 1→0.75→1.1→1

**수집 완료 체크 (`chk0/1/2`):**
- 클릭 후 등장 (scale 0→1.2→1)
- 흰색 체크마크 SVG
- 다음 사이클까지 유지, 100%에서 리셋

**장바구니 배지 (`badgeAppear`):**
- 첫 클릭 후 등장 (18%), 각 클릭 시 바운스 (scale 1.25)
- 88~92%에서 페이드아웃
- 카운트 (`badgeCount`): `0` → `1` (20%) → `2` (46%) → `3` (72%)

### 버튼 3개 (세로 배치, gap `spacing['2']`)

**1. "프로그램 설치하기" (Primary)**
- 높이: `52px`, `colors.primary` 배경, 흰색 텍스트

## 온보딩 완료 화면 (showComplete)

**트리거:** Phase done "시작하기" 클릭 또는 Phase idle "나중에 설치할게요" 클릭.

**전체 화면 오버레이:**
- `position: fixed`, `inset: 0`, `zIndex: 50`
- 배경: `colors.bg.surface`
- 진입: `fadeIn 0.35s cubic-bezier(0.16, 1, 0.3, 1)`

### 체크 애니메이션 (SVG)

- 크기: 72x72
- 원형 라인: `stroke-dasharray: 207`, `stroke-dashoffset: 207 → 0` (`checkCircle 0.55s`, 0.1초 딜레이)
- 체크마크 폴리라인: `stroke-dasharray: 50`, `stroke-dashoffset: 50 → 0` (`checkMark 0.35s`, 0.6초 딜레이)
- 색상: `colors.primary`, strokeWidth 2.5 (원) / 3 (체크)
- 등장: `riseUp 0.5s cubic-bezier(0.16, 1, 0.3, 1) 0.05s both`

### 완료 텍스트

- 제목: `"기본 설정이 완료됐어요"` — 0.3초 딜레이 `riseUp`
  - `font.size['2xl-']`, `font.weight.extrabold`, `letterSpacing: -0.5px`
- 설명: `"아래 설정은 이후에 언제든 변경할 수 있어요."` — 0.4초 딜레이

### 설정 요약 리스트 (3행)

| 라벨 | 설명 (desc) |
|------|------------|
| `Qoo10 JP 연동` | `{storeName}` 또는 `"연동 완료"` |
| `기본 정보` | `연락처 · 출하지 · 배송비 · 마진 설정 완료` |
| `프로그램 설치` | Phase done이면 `"설치 완료"`, 아니면 `"나중에 설치"` |

- 각 행: 좌측 파란 체크 아이콘 (16x16 SVG) + 라벨 (`font.weight.semibold`) + 우측 설명 (`colors.text.muted`)
- 테두리: `1px solid colors.bg.subtle`, `radius.xl`
- 등장: 0.5초 딜레이 `riseUp`

### "수집 시작하기" 버튼

- 높이: `54px`, `colors.primary` 배경
- `font.size['base+']`, `font.weight.bold`, `letterSpacing: -0.2px`
- `ArrowRight` 아이콘 (18px)
- 호버: 배경 `colors.primaryDark`
- 등장: 0.6초 딜레이 `riseUp`
- 클릭: `transitionTo('/sourcing')` → SC-01 이동

## 동작 정의

### 프로그램 설치
- Given: Phase idle
- When: "프로그램 설치하기" 클릭
- Then:
  1. 기존 타이머 클리어
  2. `window.open(WEBSTORE_URL, '_blank')` — 크롬 웹스토어 새 탭
  3. `phase = 'waiting'`
  4. 10초 타이머 시작 → 만료 시 `phase = 'done'` + `sessionStorage ext_installed = 'true'`

### 나중에 설치
- Given: Phase idle
- When: "나중에 설치할게요" 클릭
- Then: `showComplete = true` → 완료 화면 (설치 건너뜀)

### 웹스토어 다시 가기
- Given: Phase waiting
- When: "웹스토어 다시 가기" 클릭
- Then: `window.open(WEBSTORE_URL, '_blank')` — 크롬 웹스토어 재열기

### 시작하기
- Given: Phase done
- When: "시작하기" 클릭
- Then: `showComplete = true` → 완료 화면

### 수집 시작
- Given: 완료 화면
- When: "수집 시작하기" 클릭
- Then: `transitionTo('/sourcing')` → SC-01 이동

### 이전으로
- Given: Phase idle
- When: "이전으로" 클릭
- Then: `transitionTo('/basic-info')` → S-03 이동
