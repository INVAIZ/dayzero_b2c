자동 수집 설정 화면을 사용성 테스트 피드백에 맞춰 개선했습니다. 새로 만들 때는 3단계로 나눠 하나씩 설정하고, 수정할 때는 한 화면에서 바로 고칠 수 있도록 바꿨습니다. 설정 완료 후 불필요한 확인 창 없이 바로 목록으로 돌아가며, 목록 카드에 수집 시간이 바로 보여 현재 상태를 한눈에 파악할 수 있습니다.

## 의사결정 기록

### 스텝 바이 스텝 vs 한 번에 보기 분리
- **결정**: 신규 생성은 3단계 위저드, 수정은 단일 화면
- **고려한 대안**: 모든 경우 단일 화면 / 모든 경우 위저드
- **선택 이유**: UT에서 "설정하면 바로 수집 시작되는 줄 알았다"는 피드백 — 단계 분리로 인지 부하 해소
- **트레이드오프**: 코드 복잡도 증가 (renderStep1~3 + renderEditMode)

### 완료 모달/토스트 제거
- **결정**: 저장 후 바로 목록으로 navigate
- **고려한 대안**: 완료 모달 → 토스트 → 제거
- **선택 이유**: 모달은 추가 클릭 필요, 토스트는 위치 불안정. 바로 목록 이동이 가장 자연스러움
- **트레이드오프**: 저장 완료 피드백이 없음 — 목록에 새 카드가 보이는 것 자체가 피드백

### 수정 모드에서 쇼핑몰 변경 불가
- **결정**: 수정 시 카테고리/시간만 변경 가능, 쇼핑몰 변경은 새로 생성
- **선택 이유**: 쇼핑몰 변경 시 카테고리 트리가 완전히 달라져 혼란 발생
- **트레이드오프**: 쇼핑몰 변경 원하면 삭제 후 재생성 필요
